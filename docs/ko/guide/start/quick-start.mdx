# 빠른 시작

Lynx 문서에 오신 것을 환영합니다! Lynx 프로젝트를 생성하고 개발을 시작해보겠습니다.

## 시스템 요구사항

- [Node.js 18](https://nodejs.org/en) 이상.
  - TypeScript를 설정으로 사용할 때는 Node.js 18.19가 필요합니다.

## 설치

<Steps>

### 새 Lynx 프로젝트 생성

Lynx 프로젝트를 빌드하기 위해 Rspeedy(Rspack 기반 Lynx 빌드 도구)를 사용합니다.

모든 것을 자동으로 설정해주는 [`create-rspeedy`](https://npmjs.org/package/create-rspeedy)를 사용하여 새 프로젝트를 시작하는 것을 권장합니다. 프로젝트를 생성하려면 다음을 실행하세요:

<PackageManagerTabs command="create rspeedy@latest" />

프롬프트를 완료하면 `create-rspeedy`가 프로젝트 이름으로 폴더를 생성합니다.

### Lynx Explorer 준비

Lynx Explorer는 Lynx를 빠르게 시험해볼 수 있는 샌드박스입니다.

<PlatformTabs queryKey="explorer-platform">

<PlatformTabs.Tab platform="ios-simulator">
현재 iOS 시뮬레이터용 사전 빌드 바이너리만 제공합니다. 실제 iOS 기기에서 Lynx Explorer를 실행하려면 소스에서 빌드해야 합니다. [iOS용 Lynx Explorer 빌드](https://github.com/lynx-family/lynx/tree/develop/explorer/darwin/ios) 가이드를 참조하세요.

:::info
커뮤니티 기여자가 게시한 Lynx Explorer 버전이 [App Store](https://apps.apple.com/us/app/lynx-go-dev-explorer/id6743227790)에서도 제공됩니다.
이 버전은 Lynx 팀이 검토하거나 유지 관리하지 않지만, 개발자가 iOS에서 Lynx를 더 편리하게 시험할 수 있도록 해준 커뮤니티에 감사드립니다.
:::

1. **Xcode 설치**

Mac App Store를 열고 [Xcode](https://apps.apple.com/us/app/xcode/id497799835)를 검색한 다음 설치를 클릭하세요(이미 있다면 업데이트).

2. **<div id="download-lynx-explorer,ios-simulator-platform=macos-arm64,explorer-platform=ios-simulator">LynxExplorer 다운로드</div>**

<PlatformTabs queryKey="ios-simulator-platform">

<PlatformTabs.Tab platform="macos-arm64">

[`LynxExplorer-arm64.app.tar.gz`](https://github.com/lynx-family/lynx/releases/latest/download/LynxExplorer-arm64.app.tar.gz)를 다운로드하세요.

그런 다음 다운로드한 아카이브를 압축 해제합니다:

```bash
mkdir -p LynxExplorer-arm64.app/
tar -zxf LynxExplorer-arm64.app.tar.gz -C LynxExplorer-arm64.app/
```

3. **시뮬레이터에 LynxExplorer 설치**

Xcode를 열고 Xcode 메뉴에서 **Open Developer Tool**을 선택합니다. **Simulator**를 클릭하여 시뮬레이터를 실행합니다. "LynxExplorer-arm64.app"을 시뮬레이터로 드래그합니다.

</PlatformTabs.Tab>

<PlatformTabs.Tab platform="macos-intel">

[`LynxExplorer-x86_64.app.tar.gz`](https://github.com/lynx-family/lynx/releases/latest/download/LynxExplorer-x86_64.app.tar.gz)를 다운로드하세요.

그런 다음 다운로드한 아카이브를 압축 해제합니다:

```bash

```

3. **시뮬레이터에 LynxExplorer 설치**

Xcode를 열고 Xcode 메뉴에서 **Open Developer Tool**을 선택합니다. **Simulator**를 클릭하여 시뮬레이터를 실행합니다. "LynxExplorer-x86_64.app"을 시뮬레이터로 드래그합니다.

현재 Harmony 시뮬레이터용 사전 빌드 바이너리만 제공합니다. 실제 Harmony 기기에서 Lynx Explorer를 실행하려면 소스에서 빌드해야 합니다. [Harmony용 Lynx Explorer 빌드](https://github.com/lynx-family/lynx/tree/develop/explorer/harmony) 가이드를 참조하세요.

2. **<div id="download-lynx-explorer">Lynx Explorer 다운로드</div>**

또는 [Harmony용 Lynx Explorer 빌드](https://github.com/lynx-family/lynx/tree/develop/explorer/harmony) 가이드를 따라 소스에서 빌드할 수 있습니다.

3. **시뮬레이터에 Lynx Explorer 설치**

```bash
hdc install lynx_explorer-default-unsigned.hap
```

### 개발 시작

1. 생성된 프로젝트로 이동합니다:

```bash
cd <project-name>
```

2. 패키지 매니저로 NPM 의존성을 설치합니다:

<PackageManagerTabs command="install" />

3. 개발 서버를 시작하려면 다음을 실행하세요:

<PackageManagerTabs command="run dev" />

터미널에 QR 코드가 표시됩니다. Lynx Explorer 앱으로 스캔하거나 시뮬레이터를 사용하는 경우 번들 URL을 복사하여 Lynx Explorer 앱의 "Enter Card URL" 입력란에 붙여넣고 "Go"를 누르세요.

4. 첫 번째 변경하기

코드 편집기에서 `src/App.tsx` 파일을 열고 변경하세요.

Lynx Explorer의 UI가 자동으로 업데이트되는 것을 볼 수 있습니다.
